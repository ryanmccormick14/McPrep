# McPrep Academy — Project Memory

## What This Project Is
A marketing website for McPrep Academy, a gamified SAT/ACT test prep 
business founded by Ryan McCormick. The site targets two audiences:
parents of high school students (primary) and competitive high school 
students (secondary).

## Business Info

## Program Details
- Format: Live on Zoom, small group
- Schedule: Mondays, 7:00–9:00 PM PT
- Sessions: 3 sessions per boot camp, 2 hours each (6 hrs total)
- Price: $180 per boot camp ($30/hr)

## Tech Stack
- Language: Plain HTML, CSS, JavaScript (single file or multi-file)

## Deployment Workflow
- Edit code in Cursor
- Commit changes in Source Control panel
- Push to GitHub → Vercel auto-deploys
- Never manually upload files to Vercel

## Brand & Design
- Aesthetic: Clean, professional, modern — think high-quality productivity 
  app meets competitive gaming interface
- Avoid: childish, cartoonish, generic tutoring imagery
- Colors: Academic credibility with subtle gaming energy
- Logo: McPrep logo with rocket icon — "Mc" in black, "Prep" in blue gradient
- Images: High school students collaborating, competing, celebrating
- Text: Avoid hyphens "—" as it makes the website appear AI generated.

## Coding Rules
- Keep all CSS in <style> blocks or external .css files — no inline styles
- All layouts must be fully responsive (mobile-first)
- Hide non-essential table columns on screens under 600px
- Use DM Sans as the primary font (import from Google Fonts)
- Rounded containers: border-radius 12px standard
- No JavaScript frameworks — vanilla JS only (for now)
- Always preserve existing content when editing — never delete sections 
  unless explicitly asked
- Ask before making major layout changes
- As we build and expand this project, continuously update this claude.md file to reflect new features, file structure changes, conventions, and decisions made along the way — so it stays accurate and useful as the project grows.
